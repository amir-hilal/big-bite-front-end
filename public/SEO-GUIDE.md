# SEO & Social Media Setup

## Open Graph Image (og-image.jpg)

For best social media previews, create and add an Open Graph image to `/public/og-image.jpg`

**Specifications:**
- Size: 1200x630 pixels (required for Facebook/LinkedIn)
- Format: JPG or PNG
- File size: Under 8MB
- Content: Should showcase your brand/food

**What to include:**
- Your logo
- Food images
- Business name "Big Bite Concession"
- Tagline or key message
- Vibrant, eye-catching design

**Tips:**
- Use high-quality images
- Keep text readable at small sizes
- Test on both light and dark backgrounds
- Ensure safe zones (no important content at edges)

## Update Domain

Once you have a domain, replace `https://bigbiteconcession.com/` in these files:
1. `/index.html` - Update all og:url, twitter:url, and canonical links
2. Each page's SEO component uses relative paths, so no changes needed there
